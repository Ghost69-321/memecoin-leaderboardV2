If you see 401 Unauthorized:

- Try switching **Auth header mode** in the sidebar: Bearer → X-API-Key → LC-API-Key.
- Try toggling **Use /public segment** off (some accounts/endpoints differ).
- Use the **Test API now** button to confirm.
